# AAPFS
An Apple Pie From Scratch

A basic elastic collision simulator with springs.

## Install
`python -m venv`
`source /bin/activate`
`pip install -r requirements.txt`

## Run
`python gui.py`
Click "Snap to 2D"
Click "Solve"
Click "Play"
"Analysis" Button graphs accelerations, velocities
